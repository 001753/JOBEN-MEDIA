🔜 Langkah Selanjutnya Sebelum Go Live
1. Login ke Strapi Admin — gunakan workflow "Start CMS" lalu buka /admin:

Buat role Editor dan Penulis dengan permissions yang tepat
Generate API Token read-only untuk frontend (Settings → API Tokens)
Tambah sekret STRAPI_API_TOKEN di Replit Secrets setelah token dibuat
2. Isi Konten Awal:

Category: Politik, Ekonomi, Olahraga, Teknologi, Hiburan, dll
Author: profil redaksi
Page: tentang-kami, kebijakan-privasi, pedoman-siber, kontak
Artikel perdana → setelah dipublish, homepage langsung hidup!
3. Untuk Production di cPanel:

Strapi: jalankan dari root dengan node server.js (PORT dari Passenger)
Frontend: jalankan dari frontend/ dengan node server.js (PORT dari Passenger)
Ubah env vars production: STRAPI_API_URL=https://cms.news.jobenapp.cloud, NEXTJS_REVALIDATION_URL=https://news.jobenapp.cloud/api/revalidate
